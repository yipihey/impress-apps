# ImpressKit

Shared Swift package providing access to impress Rust crates via UniFFI.

## Status

Planned. Will include:

- `ImpressDomain` - Domain types (Publication, Author, etc.)
- `ImpressBibtex` - BibTeX parsing
- `ImpressIdentifiers` - DOI/arXiv extraction
- `ImpressCollab` - Collaboration infrastructure
