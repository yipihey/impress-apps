# imprint

Collaborative academic writing app using Typst.

## Status

In development. See [ADR-002](./ADR-002-imprint-architecture.md) for architecture details.

## Features

- Typst-based document authoring
- Real-time collaboration with CRDT sync
- Three editing modes: Direct PDF, Split View, Text Only
- Citation integration with imbib
- LaTeX import/export for journal submission
