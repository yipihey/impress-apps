# Keyboard Shortcuts

imBib is designed for efficient keyboard-driven workflows. This page lists all available shortcuts.

---

## Navigation

### Sidebar

| Action | Shortcut |
|--------|----------|
| Focus sidebar | `Cmd-1` |
| Go to Inbox | `Cmd-Shift-I` |
| Go to All Publications | `Cmd-Shift-A` |
| Toggle sidebar | `Cmd-S` |
| Cycle libraries | `Cmd-[` / `Cmd-]` |

### Paper List

| Action | Shortcut |
|--------|----------|
| Focus list | `Cmd-2` |
| Next paper | `↓` or `J` |
| Previous paper | `↑` or `K` |
| First paper | `Cmd-↑` or `Home` |
| Last paper | `Cmd-↓` or `End` |
| Page down | `Space` or `Page Down` |
| Page up | `Shift-Space` or `Page Up` |
| Select all | `Cmd-A` |
| Extend selection | `Shift-↓` / `Shift-↑` |

### Detail View

| Action | Shortcut |
|--------|----------|
| Focus detail | `Cmd-3` |
| Info tab | `Cmd-4` |
| BibTeX tab | `Cmd-5` |
| PDF tab | `Cmd-6` |
| Notes tab | `Cmd-7` |

---

## Search

| Action | Shortcut |
|--------|----------|
| Focus search field | `Cmd-F` or `/` |
| Quick Lookup (by ID) | `Cmd-Shift-L` |
| Execute search | `Enter` |
| Clear search | `Escape` |
| Previous search | `Cmd-Shift-F` |

---

## Paper Actions

### Reading

| Action | Shortcut |
|--------|----------|
| Toggle read status | `R` |
| Mark as read | `Shift-R` |
| Mark as unread | `Cmd-Shift-R` |
| Mark all read | `Cmd-Opt-R` |

### Opening

| Action | Shortcut |
|--------|----------|
| Open PDF | `Cmd-O` or `Enter` |
| Open PDF in external app | `Cmd-Shift-O` |
| Open notes | `Cmd-N` |
| Open in browser (DOI) | `Cmd-Shift-D` |
| Open in browser (arXiv) | `Cmd-Shift-X` |
| Open in browser (ADS) | `Cmd-Shift-B` |
| View references | `Cmd-Shift-R` |
| View citations | `Cmd-Shift-C` |

### Editing

| Action | Shortcut |
|--------|----------|
| Edit BibTeX | `Cmd-E` |
| Save changes | `Cmd-S` |
| Cancel edit | `Escape` |
| Edit cite key | `Cmd-K` |

---

## Clipboard

| Action | Shortcut |
|--------|----------|
| Copy BibTeX | `Cmd-C` |
| Copy cite key | `Cmd-Shift-K` |
| Copy citation | `Cmd-Shift-C` |
| Copy DOI | `Cmd-D` |
| Cut (for move) | `Cmd-X` |
| Paste | `Cmd-V` |

---

## Import & Export

| Action | Shortcut |
|--------|----------|
| Import BibTeX file | `Cmd-I` |
| Import from identifier | `Cmd-Shift-L` |
| Add to library | `Cmd-Shift-I` |
| Export selection | `Cmd-Shift-E` |

---

## Inbox Triage

| Action | Shortcut |
|--------|----------|
| Star/unstar paper | `S` |
| Archive to library | `A` |
| Dismiss (remove) | `D` |
| Next inbox item | `J` or `↓` |
| Previous inbox item | `K` or `↑` |

---

## PDF Viewer

| Action | Shortcut |
|--------|----------|
| Next page | `→` or `J` or `Space` |
| Previous page | `←` or `K` or `Shift-Space` |
| First page | `Home` or `Cmd-↑` |
| Last page | `End` or `Cmd-↓` |
| Go to page | `Cmd-G` |
| Zoom in | `Cmd-+` or `Cmd-=` |
| Zoom out | `Cmd--` |
| Actual size (100%) | `Cmd-0` |
| Fit to window | `Cmd-9` |
| Fit width | `Cmd-8` |
| Find in PDF | `Cmd-F` |
| Find next | `Cmd-G` |
| Find previous | `Cmd-Shift-G` |

---

## Library Management

| Action | Shortcut |
|--------|----------|
| New library | `Cmd-Opt-N` |
| New collection | `Cmd-Shift-N` |
| New smart search | `Cmd-Opt-S` |
| Delete selection | `Delete` or `Backspace` |
| Delete (with confirm) | `Cmd-Delete` |

---

## Window & View

| Action | Shortcut |
|--------|----------|
| Toggle sidebar | `Cmd-S` |
| Toggle detail pane | `Cmd-D` |
| Show unread only | `Cmd-U` |
| Show papers with PDFs | `Cmd-P` |
| Refresh | `Cmd-R` |
| Open settings | `Cmd-,` |
| Show console | `Cmd-Opt-C` |

---

## Smart Search

| Action | Shortcut |
|--------|----------|
| New smart search | `Cmd-Opt-S` |
| Execute smart search | `Cmd-R` |
| Edit smart search | `Cmd-E` (when selected) |

---

## Customizing Shortcuts

Keyboard shortcuts are currently not customizable. We plan to add this in a future release.

If you have suggestions for additional shortcuts, please [open an issue](https://github.com/yipihey/imbib/issues).
