# Manuscript Tracking

Track your own papers through the publication process and integrate with imprint for writing.

---

## Overview

imbib can track manuscripts you're writing:

- **Status tracking** - From drafting through publication
- **Citation management** - Track papers you cite
- **Version control** - Organize submission and revision documents
- **imprint integration** - Seamless writing workflow

---

## Manuscript Status

| Status | Description |
|--------|-------------|
| **Drafting** | Initial writing |
| **Submitted** | Sent to journal |
| **Under Review** | Being reviewed |
| **In Revision** | Revising |
| **Accepted** | Accepted for publication |
| **Published** | Final publication |

### Changing Status

Select manuscript > **Paper > Manuscript Status** > Choose status

Manuscripts appear in sidebar collections: My Manuscripts, Active, Completed.

---

## Citation Tracking

### Linking Cited Papers

1. Select your manuscript
2. Go to **Citations** in Info tab
3. Click **Add Citation**
4. Select papers from your library

### Exporting Bibliography

**Paper > Export > Bibliography...** exports all cited papers as `.bib`.

---

## Document Versions

Organize versions with tags:

| Tag | Use For |
|-----|---------|
| **Submission v1/v2/v3** | Submitted versions |
| **Revision R1/R2/R3** | Revision rounds |
| **Referee Report** | Reviewer feedback |
| **Response Letter** | Your responses |
| **Final Accepted** | Accepted manuscript |

Right-click attachments to set tags.

---

## imprint Integration

### Linking Documents

1. Select manuscript
2. **Paper > Link to imprint Document**
3. Choose or create `.imprint` document

### Opening in imprint

Click the imprint icon or use **Paper > Open in imprint**.

### Citation Workflow

In imprint: **Cmd+Shift+K** to insert citations from imbib.

---

## Workflows

### Initial Submission

1. Create manuscript (Drafting)
2. Add cited papers
3. Change to Submitted
4. Attach submission PDF

### Handling Reviews

1. Add referee reports
2. Change to In Revision
3. Attach response letter
4. Submit revision
5. Change back to Submitted

### After Acceptance

1. Change to Accepted
2. Attach final manuscript
3. Change to Published when live
4. Add DOI
